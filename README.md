# 🌐 HTML Basics — Visual Reference Guide

A high-performance, interactive visual guide to HTML fundamentals, built as a reference for the **freeCodeCamp** curriculum. This project serves as a "cheat sheet" for beginners to understand elements, attributes, structure, and modern web best practices.

## 🚀 Key Features

* **Interactive Reference**: Visual cards for common elements like Headings, Lists, and Semantic Layout.
* **Click-to-Copy Snippets**: Custom JavaScript allows users to instantly copy code blocks for their own projects.
* **Modern Design**: A sleek "Dark Mode" aesthetic built with CSS variables for high-contrast readability.
* **Scroll Reveal Animations**: Smooth entry animations using the Intersection Observer API for an engaging user experience.
* **Mobile Optimized**: Fully responsive layout that works on desktops, tablets, and phones.

## 🛠️ Built With

* **HTML5**: Semantic structure using `<main>`, `<section>`, and `<article>`.
* **CSS3**: Modern styling with custom properties (variables), Flexbox, and CSS Grid.
* **JavaScript (ES6)**: Functional programming for scroll reveals, active navigation highlighting, and clipboard interaction.

## 📂 Project Structure

* `index.html`: The core content and semantic structure of the guide.
* `styles.css`: The visual engine, including the color palette and responsive media queries.
* `script.js`: The interactivity layer for animations and utility features.

## 📖 What This Guide Covers

* **Document Structure**: The essential HTML boilerplate (`DOCTYPE`, `head`, `body`).
* **Media & Multimedia**: Proper implementation of `<img>`, `<video>`, `<audio>`, and `<iframe>`.
* **Path Logic**: Clear distinctions between Absolute and Relative file paths.
* **SEO Basics**: Implementation of Meta descriptions and Open Graph tags for social sharing.
